# threejs-practice
Learning three.js

## Installing

1. Clone Repository

```bash
git clone https://github.com/anand-2020/threejs-practice.git
```

2. CD into folder

```bash
cd threejs-practice
```

3. Install TypeScript

```bash
npm install -g typescript
```

4. Install dependencies

```bash
npm install
```

5. Start it

```bash
npm run dev
```

6. Visit [http://127.0.0.1:8080](http://127.0.0.1:8080)
